### Botones de la Calculadora

- **Números:** 0 - 9
- **Operaciones:**
  - `+` Suma
  - `-` Resta
  - `*` Multiplicación
  - `/` División
- **Otros:**
  - `.` Decimal
  - `=` Igual
  - `C` / `AC` Limpiar todo
  - `DEL` Borrar último dígito
  - `+/-` Cambiar signo
  - `%` Porcentaje

### Estructura de la calculadora

[ Input de historial ] [ Operacion]
[ Input de resultado ]
